    Datasets

    This page lists the datasets used in the project and provides direct access to their original sources. All datasets are publicly available and were selected to allow consistent cross-country comparison of economic outcomes, innovation performance, and policy-relevant inputs.

    GDP per Capita

    World Bank dataset providing GDP per capita in current US dollars. This dataset is used as the core outcome variable to measure income differences across countries.

    https://data.worldbank.org/indicator/NY.GDP.PCAP.CD

    Global Innovation Index (GII)

    Composite index published by the World Intellectual Property Organization (WIPO), measuring innovation performance across countries using multiple input and output indicators. This dataset is used as the key innovation outcome variable in the project.

    https://www.wipo.int/en/web/global-innovation-index

    R&D Expenditure (% of GDP)

    World Bank dataset reporting research and development expenditure as a percentage of GDP. This dataset is used as the primary policy-relevant input to examine whether innovation can be influenced through investment.

    https://data.worldbank.org/indicator/GB.XPD.RSDV.GD.ZS
